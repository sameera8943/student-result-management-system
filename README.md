# 🎓 Student Result Management System

A Python-based **Student Result Management System** that helps manage student academic records efficiently. The application allows users to store student details, calculate total marks, percentage, and grades automatically, and save all records in a CSV file using Python file handling.

---

## 📌 Project Overview

The Student Result Management System is a console-based Python application developed to simplify the process of managing student results. It provides an easy-to-use menu-driven interface for adding and storing student information while automatically calculating academic performance.

This project demonstrates the use of Python programming concepts such as file handling, functions, loops, conditional statements, and CSV data management.

---

## ✨ Features

- ➕ Add new student records
- 📄 Store records in a CSV file
- 🧮 Automatically calculate total marks
- 📊 Calculate percentage
- 🏆 Generate grades automatically
- 💾 Save data permanently using CSV
- 🖥️ Simple menu-driven interface
- 🔄 Easy to modify and extend
